# Finance Advisor

A premium landing page designed to introduce users to a Finance Advisor service.

## Overview
This project provides a simple, high-end interface with a single objective: connecting users with a finance advisor. It explains how the service helps users intelligently plan and manage their money.

## Features
- **Premium Design**: Modern dark theme with smooth gradients and subtle animations.
- **Single CTA**: A prominent "Talk to Finance Advisor" button for high conversion.
- **Responsive**: Works seamlessly across desktop and mobile devices.

## How to Use
1. Open the project folder on your Mac.
2. Double-click the `index.html` file to view the landing page in your default browser.
3. Click the **Talk to Finance Advisor** button to be redirected to the advisor platform.

## File Structure
- `index.html`: The main structure of the landing page.
- `style.css`: The visual styling and animations.
- `script.js`: Logic for the redirect functionality.
